# PW_Skills_SQL_Assignments
## Project Overview
This Repository contains PW Skills SQL Assignments that I completed as part of my PW Skills Data Analytics Course. It covers all the basics to advanced level SQL concepts that are essential for cleaning, manipulating and analyzingthe data using SQL.
## Tools & Technologies Used
1 . I used MySQL Workbench to execute and test SQL queries.

2 . Github ( to upload , share and organize my SQL Assignments )
## Concepts Covered
### Basic SQL Queries   
- Used `SELECT` , `WHERE` , `DISTINCT` , `ORDER BY` and `LIMIT` to retrieve and filter data.
- Applied sorting and filtering techniques to extract meaningful business insights from dataset.
### Aggregate Functions 
- Worked with `COUNT`, `SUM`, `AVG`, `MIN`, and `MAX` to summarize data.
- Used `GROUP BY` and `HAVING` to perform grouped analysis on datasets.
### Joins
- Implemented `INNER JOIN`, `LEFT JOIN`, `RIGHT JOIN`, and `FULL JOIN` to combine data from multiple tables.
- Understood the importance of primary and foreign keys in relational databases.
### Subqueries
- Wrote nested queries to perform advanced filtering and conditional data retrieval.
- Used subqueries in `SELECT`, `FROM`, and `WHERE` clauses.
### Window Functions
- Applied functions like `ROW_NUMBER()`, `RANK()`, and `DENSE_RANK()` for advanced analytical queries.
- Performed partition-based calculations using `OVER()` clause.
### Data Cleaning & Manipulation
- Handled missing values and duplicates using SQL queries.
- Used string functions and conditional statements for data transformation.
- Practiced updating and modifying records using `UPDATE` and `DELETE`.
### Database Concepts
- Developed understanding of primary keys, foreign keys, and relationships between tables.
- Learned how structured query language (SQL) is used to manage and analyze relational databases.
## Assignment Tasks
- Solved Daily Practice Problems and Scenario Based Questions where I was required to use the above mentioned Functions and everthing else which I have learned in Excel.
- Gained hands on experience in writing optimized and readable queries for data analysis tasks.
